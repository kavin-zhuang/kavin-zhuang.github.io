编码规范

2016.11.1, by Jinfeng Zhuang kavin.zhuang@gmail.com

  • 编码格式：GBK

  • 将Tab键转换成两个空格

  • 最大行长度80：原则上

  • 文件声明:

      /*
   * filename.c
   * Copyright (C) 2016.11, jinfeng <kavin.zhuang@gmail.com>
   * 
   * descriptions
   */

  • 文件分段描述：就像从左边长出一个柱子，稳定的结构，适用于分割

      /*
   *==============================================================================
   * Macros
   *==============================================================================
   */

  • 注释：双斜杠用于临时注释，后期去掉

      /* note */

  • 函数风格：

      void foo(int x)
  {
    if(x) {
      x++;
    }
  }
